# Install and Run Webpack

## TLDR

1. Run `npm init`
2. Install Webpack as a dependency
3. Write /src/index.js
4. Run webpack
5. Inspect /dist/index.js
6. Update package.json with build command

# Work Log

- `yarn add webpack`
- `yarn add webpack-cli`
- create **script** in `package.json` for `npm run build`
- create `/src/index.js`
- run `npm run build`
- run your script `node dist/main.js`
